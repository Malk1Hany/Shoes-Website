# 👟 Shoes Website

A modern and responsive shoe store website built using pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

![Preview](./preview.png)

## 🌐 Live Demo

🔗 [malk1hany.github.io/Shoes-Wedsite](https://malk1hany.github.io/Shoes-Website/)

---

## 📌 Project Overview

A front-end e-commerce website for showcasing and selling shoes online. The site features a clean, attractive UI with smooth navigation across multiple sections including a product gallery, contact form, and more.

---

## ✨ Features

- 🏠 **Hero Section** — Eye-catching landing area with a call-to-action
- 👟 **Product Showcase** — Display of featured shoes with details
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- 🎯 **Smooth Scrolling** — Seamless navigation between sections
- 📬 **Contact Section** — Easy way for customers to get in touch
- 🎨 **Modern UI** — Clean design with attractive animations

---

## 🛠️ Technologies Used

| Technology | Purpose |

## 🏁 Getting Started

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Malk1Hany/Shoes-Wedsite.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Shoes-Wedsite
   ```

3. **Open in your browser**

   Simply open `index.html` in any browser — no installation needed!
